# Web Application Vulnerability Research Lab

This repository documents my hands-on testing of common web application vulnerabilities in a controlled lab environment (DVWA, OWASP ZAP).

## Vulnerabilities Covered

- SQL Injection
- Cross-Site Scripting (XSS)
- Cross-Site Request Forgery (CSRF)
- Insecure File Upload

## Lab Environment
- OWASP ZAP
- DVWA (Damn Vulnerable Web Application)
- Kali Linux
